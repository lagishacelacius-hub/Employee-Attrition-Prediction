Employee Attrition Prediction Using Machine Learning

Project Description

Employee attrition prediction is a Machine Learning project that predicts whether an employee is likely to leave an organization based on different employee-related factors.

This project helps HR departments understand employee patterns, identify employees who may leave, and take preventive actions to improve retention.

Objective

* Analyze employee data to find factors affecting attrition.
* Build a machine learning model to predict employee turnover.
* Assist HR teams in making data-driven decisions.

Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

Machine Learning Model

The project uses classification algorithms such as:

* Logistic Regression
* Decision Tree
* Random Forest Classifier

Dataset Features

The dataset includes employee details such as:

* Age
* Job Role
* Monthly Income
* Job Satisfaction
* Work Experience
* Overtime
* Work-Life Balance
* Attrition

Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Preprocessing
5. Handling Class Imbalance using SMOTE
6. Model Training
7. Model Evaluation
8. Employee Attrition Prediction

Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

Application

This project can help HR departments identify employees who have a higher probability of leaving and develop strategies to improve employee satisfaction and retention.

Future Improvements

* Deploy the model as a web application using Streamlit.
* Add more employee-related factors.
* Improve prediction performance using advanced models.
